# hyperzod-ui-ordering-design-token

Design tokens generated by [Tokener](https://www.figma.com/community/plugin/tokener).

## Installation

```bash
npm install hyperzod-ui-ordering-design-token
```

## Usage

**JavaScript/TypeScript:**
```javascript
import tokens from 'hyperzod-ui-ordering-design-token';
```

**CSS:**
```css
@import 'hyperzod-ui-ordering-design-token/css';
```

**Tailwind:**
```javascript
const tokens = require('hyperzod-ui-ordering-design-token/tailwind');
```
